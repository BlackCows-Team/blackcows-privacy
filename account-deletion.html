📧 계정 삭제 방법

소담소담 앱의 계정 삭제를 원하시면 아래 이메일로 요청해주세요.

📨 계정 삭제 요청하기

📝 요청 시 포함할 정보

  • 사용자 이름: 앱에서 사용하는 이름
  • 이메일 주소: 가입 시 사용한 이메일
  • 삭제 사유 (선택사항)

⏰ 처리 시간

계정 삭제 요청은 영업일 기준 3-5일 이내에 처리됩니다.

🗑️ 삭제되는 데이터

  • 개인정보 (이름, 이메일, 비밀번호)
  • 목장 정보 및 설정
  • 젖소 관리 데이터
  • AI 챗봇 대화 기록
  • 모든 앱 사용 기록

⚠️ 주의사항

계정 삭제 후에는 데이터 복구가 불가능합니다. 신중하게 결정해주세요.

📞 문의

기타 문의사항이 있으시면 언제든 연락주세요.
